ImagingTriage User Guide

This is the English help page for ImagingTriage. Detailed instructions on how to use the application will be provided here.

Features:

  • Process images based on XMP metadata (ratings and labels).
  • Organize files into subfolders.
  • Gather files back from subfolders.
  • Configurable supported file extensions.

How to Use:

  1. Select the folder containing your images.
  2. Choose processing options (e.g., inhibit move unrated).
  3. Click "Process" to start.

For more information, please refer to future updates.
