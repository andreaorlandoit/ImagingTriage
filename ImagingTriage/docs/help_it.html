ImagingTriage Guida Utente

Questa è la pagina di aiuto in italiano per ImagingTriage. Qui verranno fornite istruzioni dettagliate su come utilizzare l'applicazione.

Funzionalità:

  • Elabora immagini basandosi sui metadati XMP (valutazioni ed etichette).
  • Organizza i file in sottocartelle.
  • Raccoglie i file dalle sottocartelle.
  • Estensioni di file supportate configurabili.

Come Usare:

  1. Seleziona la cartella contenente le tue immagini.
  2. Scegli le opzioni di elaborazione (es. inibisci spostamento non valutati).
  3. Clicca su "Elabora" per iniziare.

Per maggiori informazioni, fare riferimento ai futuri aggiornamenti.
